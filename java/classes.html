Java Classes

Classes are the building blocks of Java programs. They define the structure and behavior of objects.

Example


class Car {
  String brand;
  int speed;

  void drive() {
    System.out.println(brand + " is driving at " + speed + " km/h");
  }
}

public class Main {
  public static void main(String[] args) {
    Car car = new Car();
    car.brand = "Tesla";
    car.speed = 120;
    car.drive();
  }
}
