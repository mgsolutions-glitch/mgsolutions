Java Setup

In this topic, you will learn how to install Java Development Kit (JDK) and set up your development environment for Java programming.

Step 1: Download JDK

Visit the official Oracle JDK website or OpenJDK website to download the latest version of JDK. Download JDK

Step 2: Install JDK

Run the downloaded installer and follow the on-screen instructions to install JDK on your system.

  • Windows: Run the .exe file.
  • Mac: Run the .dmg file.
  • Linux: Use your package manager or tarball.

Step 3: Configure Environment Variables

Add the bin folder of the JDK installation to your system’s PATH environment variable. This allows you to run java and javac from any directory in your terminal.

Step 4: Verify Installation

Open a terminal or command prompt and type: 

java -version
javac -version

If installed correctly, you should see the installed Java version.

Step 5: Install an IDE

Install an IDE like IntelliJ IDEA, Eclipse, or VS Code with Java extensions to make coding easier.

  • IntelliJ IDEA
  • Eclipse
  • VS Code

✅ You are now ready to start programming in Java!
