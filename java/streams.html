Java Streams API

Streams in Java provide a modern way to process collections of data using functional-style operations.

Example


import java.util.*;

public class Main {
  public static void main(String[] args) {
    List numbers = Arrays.asList(1, 2, 3, 4, 5, 6);

    numbers.stream()
           .filter(n -> n % 2 == 0)
           .map(n -> n * n)
           .forEach(System.out::println);
  }
}

This program prints squares of even numbers from the list.
