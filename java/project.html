Final Project

Apply everything you’ve learned to build a real-world Java application.

Project Idea

Build a Student Management System with the following features:

  • Store student details (name, ID, marks)
  • Use classes and objects to represent students
  • Implement inheritance (e.g., Undergraduate, Graduate)
  • Use Streams API to filter and analyze results
  • Provide a simple console menu for interaction

Sample Code Snippet


class Student {
  String name;
  int marks;
  
  Student(String name, int marks) {
    this.name = name;
    this.marks = marks;
  }
}

public class Main {
  public static void main(String[] args) {
    List students = List.of(
      new Student("Alice", 85),
      new Student("Bob", 60),
      new Student("Charlie", 92)
    );

    students.stream()
            .filter(s -> s.marks > 70)
            .forEach(s -> System.out.println(s.name));
  }
}
