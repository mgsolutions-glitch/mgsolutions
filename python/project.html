Python Final Project

This project will help you combine all concepts learned: variables, functions, OOP, and file handling.

Project Idea: Student Management System

  • Create a Student class with name, age, and ID.
  • Allow adding students, displaying details, and saving them to a file.
  • Implement functions for search and update.
  • Use inheritance to create specialized classes like GraduateStudent or UndergraduateStudent.

Sample Code Snippet

class Student:
    def __init__(self, name, age, student_id):
        self.name = name
        self.age = age
        self.student_id = student_id

students = []

def add_student(name, age, sid):
    students.append(Student(name, age, sid))

add_student("Alice", 20, 101)
add_student("Bob", 22, 102)

for s in students:
    print(f"{s.student_id} - {s.name}, {s.age}")

Save this data to a file, read it back, and display student details dynamically. This project will give you hands-on experience with Python programming.
