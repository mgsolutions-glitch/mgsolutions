Python Syntax

Python uses a clean and readable syntax which makes it easy for beginners.

Variables

name = "John"
age = 25
is_student = True

Indentation

Python uses indentation (4 spaces) to define blocks instead of braces.

if age > 18:
    print("Adult")
else:
    print("Minor")

Comments

# This is a single-line comment
"""
This is a
multi-line comment
"""

Data Types

  • Integer: 10
  • Float: 3.14
  • String: "Hello"
  • Boolean: True / False
