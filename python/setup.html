Python Setup

Python is a powerful and beginner-friendly programming language. Before you start coding, we need to set up Python on your system.

Step 1: Install Python

  • Download Python from python.org.
  • Install the latest version for your operating system.
  • Check installation using terminal/command prompt: 
    python --version

Step 2: Install Code Editor

  • Recommended: Visual Studio Code (VS Code).
  • Install Python extension for code highlighting and IntelliSense.

Step 3: Your First Python Program

print("Hello, World!")

Run it using your terminal: python filename.py

Step 4: Optional Setup

  • Set up virtual environments using python -m venv env.
  • Install packages using pip install package_name.
